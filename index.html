SUMMARY

This document contains a collection of animated gifs illustrating features of the HotUI prototype

All features shown in these demos may change drastically before shipping or may not ship at all.

Directory of videos:

  • Tweaking a color property at 60fps

Navigation Demos

Widget screenshots stay in view any time text from the build method is visible.

Screenshots of build methods are displayed at the start of a build method but stay sticky staying pinned to the top of the editor window even when the top of the build method is scrolled out of view.

Hovering over the preview highlights the widget that would be selected and clicking navigates the code editor to that widget as well as changing the overall inspector selection

Widgets that are templates stamped multiple times in an application

If multiple instances of a widget are active in the application, the screenshots shows how many elements are active and clicking on the screenshot title bar toggles between the instances and shows the selection in the inspector.

Files with multiple build methods

If a file has multiple build methods, screenshots for all build methods are displayed simplifying understanding what each build method does.

Editor Demos

Inspector selection makes it clear what widgets will be impacted by a source code change.

Selection in the inspector and all widget or device mirror windows is pivoted to show all widgets at the source location of the inspector to avoid confusion about what impact editing the selected widget will have.

Tweaking a color property at 60fps

The device updates at 60fps in this demo by tweaking render objects directly. The screenshot in the code editor updates at about 10fps which is still adequate. The color picker puts pressure on the gif encouder so ignore the odd color artifacts. The color picker is just a placeholder. A real implementation of this feature will lean heavily on material colors.

Editing an enum or enum-like property

Clicking on the widget in either of the widget previews opens the property editor. Applying the property edit applies a code editor change using the analysis server and triggers a hot reload.
